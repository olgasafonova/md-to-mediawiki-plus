## [1.0.0] - 2024-01-XX

### Added
- Tieto branding with Hero Blue (#021e57) headings
- Yellow and Hero Blue inline code styling
- WCAG AA compliant color scheme
- Smart changelog reversal (newest entries first)
- Comprehensive unit test suite
- CI/CD integration with GitHub Actions
- Makefile for build automation
- `--with-css` flag for styled output
- Examples directory with sample files

### Changed
- Complete rewrite from original converter
- Improved reliability and error handling
- Enhanced parsing logic for changelogs
- Go 1.21+ requirement

### Fixed
- Various parsing bugs from original converter
- Changelog ordering issues
- Color contrast accessibility problems
